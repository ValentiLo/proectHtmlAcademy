6.9. Вопросы по DOM и событиям для самопроверки

DOM

Что такое DOM?

DOM (Document Object Model) — это объектная модель документа, DOM описывает HTML-структуру объектами JS.

DOM — это сверхточное отражение разметки.

Как получить доступ к тегу из JS?

document.documentElement.body — длинный вариант.

documentElement соответствует html узлу.

document.body — сокращённый вариант (получение элемента с помощью ссылок для быстрого доступа).

Что, кроме тегов, является узлами DOM?

Переносы строк, табуляция, текстовое содержимое тегов, атрибуты тегов, комментарий — всё, что содержит HTML, является DOM-узлами.

Как будет выглядеть ссылка Ссылка в виде DOM-объекта? 

{
  a: {
    href: '#',
    target: '_blank',
    textContent: 'Ссылка'
  }
}

Какой узел является корнем DOM-дерева?

Корень DOM-дерева — узел document.

С помощью какого свойства можно получить все дочерние элементы узла?

Получить дочерние элементы DOM-узла можно с помощью свойства children, например, console.log(body.children) выведет все узлы, вложенные в .

JavaScript в браузере

Как работают методы querySelector и querySelectorAll? 

Метод querySelector принимает CSS-селектор и возвращает первый подходящий элемент. Он понимает любые CSS-селекторы и вызывается так:

// Поиск элемента по классу
const price = document.querySelector('.price');

     
Метод querySelectorAll работает так же, разница лишь в том, что querySelectorAll вернёт все элементы (коллекцию) с заданным селектором:

// Поиск списка элементов по классу
const productItems = document.querySelectorAll('.product__item');

Как добавить класс элементу с помощью JS? 

Добавить класс элементу можно с помощью метода classList.add:

// Ищем элемент на странице:
const product = document.querySelector('.product');

// Добавляем класс (без точки перед названием класса):
product.classList.add('product--sale');

С помощью какого метода можно создать новый элемент? 

Создать новый элемент можно с помощью метода createElement() объекта document:

// Например, создадим новый элемент списка:
const card = document.createElement('li');

Как добавить новый элемент на страницу? 

Добавить новый элемент на страницу можно, например, с помощью метода appendChild():

// Ищем элемент, в который хотим поместить новый элемент
const list = document.querySelector('.cards');

// Создаём новый элемент
const card = document.createElement('li');

// Добавляем элемент card в список list
list.appendChild(card);

Как вывести текстовое содержимое элемента или добавить новый текст элементу?

Для того, чтобы вывести текст элемента или перезаписать его, нужно использовать свойство textContent: <p>Текст</p>

const p = document.querySelector('p');
console.log(p.textContent);  // Выведет: Текст

p.textContent = 'Новый текст';
console.log(p.textContent);  // Выведет: Новый текст

События

Какой метод используется для добавления обработчика события на элемент? 

Метод addEventListener:

button.addEventListener('click', function () {
  // Инструкции
});

Что такое объект Event?

Объект Event — параметр функции-обработчика. Он передаётся браузером в эту функцию в момент наступления события. Этот объект содержит много полезных свойств и методов о событии. Например, значение положения курсора мыши, информацию о нажатой клавише или позицию скролла.

С помощью какого метода можно отменить действие браузера по умолчанию? 

Отменить действие браузера по умолчанию можно с помощью метода preventDefault():

commentForm.addEventListener('submit', function (evt) {
  evt.preventDefault(); // Отменяем отправку формы по умолчанию

  // Какие-то действия ниже
});
